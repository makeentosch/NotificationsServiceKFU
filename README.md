# Распределенный сервис отправки уведомлений

## Описание проекта

Распределенный сервис отправки уведомлений, разработанный для обработки и доставки сообщений через различные каналы связи в реальном времени. Система обеспечивает надежную доставку с поддержкой повторных отправок и возможностью гибкого добавления новых каналов коммуникации.

## Архитектура

### Основные компоненты

#### 1. **Notification Gateway**
- **REST API** для приема запросов на отправку уведомлений
- Маршрутизация сообщений по каналам
- Управление повторными отправками (retry)
- Предоставление статусов уведомлений

#### 2. **Микросервисы каналов**
- **Email Service** - отправка email через SMTP
- **SMS Service** - интеграция с SMS-шлюзами
- **Push Service** - отправка push-уведомлений
- Каждый сервис масштабируется независимо

#### 3. **Инфраструктура**
- **RabbitMQ** - асинхронная обработка сообщений
- **PostgreSQL** - хранение статусов и метаданных
- **MassTransit** + **ConnectionLib** - работа с очередями

#### 4. **Мониторинг и логирование**
- **Serilog** + **Elasticsearch** + **Kibana** - логирование
- **Prometheus** + **Grafana** - метрики производительности
- **.NET Aspire** - оркестрация и инструменты разработки
